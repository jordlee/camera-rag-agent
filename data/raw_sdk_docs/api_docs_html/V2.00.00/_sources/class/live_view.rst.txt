      - Checks to see if live-view property is readable.

    * - CrInt32u GetCode() const
      - Get the CrLiveViewPropertyCode used by this live-view property.

    * - CrPropertyEnableFlag GetPropertyEnableFlag() const
      - Get the CrPropertyEnableFlag that represents the enable status for this live-view property.
    * - CrFrameInfoType GetFrameInfoType() const

    * - CrInt8u* GetValue() const
      - | Get the value pointer.
        | This pointer is set to :ref:`CrFocusFrameInfo<cr-focus-frame-info>`, :ref:`CrMagPosInfo<cr-mag-pos-info>`, :ref:`CrFaceFrameInfo<cr-face-frame-info>`, :ref:`CrTrackingFrameInfo<cr-tracking-frame-info>` or :ref:`CrLevelInfo<cr-level-info>`.

      - | Get the time-code.
        | See below for value details
        | refs. :ref:`Time-code specifications of Monitoring<tips-time-code-specifications-of-monitoring>`
        | It is always zero for models that do not support the :ref:`ControlMonitoring()<api-control-monitoring>`.
        | Correct values are only available during recording.

      - :ref:`CrFaceFrameInfo<cr-face-frame-info>`

    * - Tracking Frame
      - | CrLiveViewProperty_TrackingFrame

    * - Level

      - :ref:`CrFramingFrameInfo<cr-FramingFrameInfo>`
