.. _command-Release:

CrCommandId_Release
=============================================

Control shutter release (S2) button.

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - CrCommandParam_Up 
    - | Up the shutter button

  * - CrCommandParam_Down 
    - | Down the shutter button
      | After executing “Down”, send “Up” to cancel the Down status.
