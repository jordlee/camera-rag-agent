.. _command-SensorCleaning:

CrCommandId_SensorCleaning
=============================================

Execute sensor cleaning.

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - CrCommandParam_Down
    - | Specify "Down" when you want to Sensor Cleaning.
      | Valid when :ref:`CrDeviceProperty_SensorCleaningEnableStatus<dp-SensorCleaningEnableStatus>` is Enable. When the Sensor Cleaning is finished, the camera restart.

| This Control Code can be executed only when the device property "Sensor Cleaning Enable Status" is Enable (0x01).
