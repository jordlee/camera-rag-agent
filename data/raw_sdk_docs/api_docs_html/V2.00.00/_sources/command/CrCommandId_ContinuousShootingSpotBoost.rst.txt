.. _command-ContinuousShootingSpotBoost:

CrCommandId_ContinuousShootingSpotBoost
=============================================

Control spot boosting button.

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - CrCommandParam_Up
    - | Be sure to specify "Up" after specifying "Down".

  * - CrCommandParam_Down
    - | Specify “Down” when you want to enable the "SpotBoost".
      | Valid when :ref:`CrDeviceProperty_ContinuousShootingSpotBoostEnableStatus<dp-ContinuousShootingSpotBoostEnableStatus>` is Enable.
