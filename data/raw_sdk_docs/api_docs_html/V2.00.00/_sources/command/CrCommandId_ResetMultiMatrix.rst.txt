.. _command-ResetMultiMatrix:

CrCommandId_ResetMultiMatrix
============================

Execute multi matrix reset.

    CrDataType: `CrDataType_UInt16`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - CrCommandParam_Up
    - | Be sure to specify "Up" after specifying "Down".

  * - CrCommandParam_Down
    - | Specify "Down" when you want to reset Multi Matrix.
