CrCommandId_CancelContentsTransfer
=============================================

Cancel content transfer

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - CrCommandParam_Up
    - | Be sure to specify "Up" after specifying "Down".

  * - CrCommandParam_Down
    - | Specify when canceling the content transfer process
      | Check the :ref:`CrDeviceProperty_ContentsTransferCancelEnableStatus<dp-ContentsTransferCancelEnableStatus>` status to see if you can cancel or not.
