.. _command-FlickerScan:

CrCommandId_FlickerScan
=======================

Execute Flicker Scan

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - CrCommandParam_Up
    - | Be sure to specify "Up" after specifying "Down".

  * - CrCommandParam_Down
    - | Specify "Down" when you want to execute Flicker Scan.
      | Valid when :ref:`CrDeviceProperty_FlickerScanEnableStatus<dp-FlickerScanEnableStatus>` is Enable.
