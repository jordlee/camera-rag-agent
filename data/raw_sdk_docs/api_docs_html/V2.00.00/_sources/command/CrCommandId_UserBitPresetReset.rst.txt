.. _command-UserBitPresetReset:

CrCommandId_UserBitPresetReset
=============================================

Execute user bit preset reset.

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - CrCommandParam_Up
    - | Be sure to specify "Up" after specifying "Down".

  * - CrCommandParam_Down
    - | Specify "Down" when you want to reset User Bit Preset.
      | Valid when :ref:`CrDeviceProperty_UserBitPresetResetEnableStatus<dp-UserBitPresetResetEnableStatus>` is Enable.

| This Control Code can be executed only when the device property "User Bit Preset Reset Enable Status" is Enable (0x01).
