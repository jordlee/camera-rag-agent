.. _command-MovieRecButtonToggle:

CrCommandId_MovieRecButtonToggle
=============================================

Control movie recording button (toggle).

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - CrCommandParam_Up
    - | Be sure to specify "Up" after specifying "Down".

  * - CrCommandParam_Down
    - | Specify “Down” when you start movie recording and stop movie recording.
      | Valid when :ref:`CrDeviceProperty_MovieRecButtonToggleEnableStatus<dp-MovieRecButtonToggleEnableStatus>` is Enable.

| Note:
| After starting movie recording, please check the movie recording status with :ref:`CrDeviceProperty_RecorderMainStatus<dp-RecorderMainStatus>`.
