.. _command-CancelFocusPosition:

CrCommandId_CancelFocusPosition
=============================================

Execute cancel absolute focus position.

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - CrCommandParam_Up
    - | Be sure to specify "Up" after specifying "Down".

  * - CrCommandParam_Down
    - | Specify "Down" when you want to cancel the Focus Position Setting.
