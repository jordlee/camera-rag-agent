.. _command-CameraSettingsReset:

CrCommandId_CameraSettingsReset
=============================================

Execute recording settings reset.

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - CrCommandParam_Down
    - | Press the setting reset button on the camera body.
      | Valid when :ref:`CrDeviceProperty_CameraSettingsResetEnableStatus<dp-CameraSettingsResetEnableStatus>` is Enable. This operation resets the camera settings and restarts the camera.

| Caution:
| The connection will be disconnected by restarting the camera. If CrReconnecting_OFF is specified for the fifth parameter of Connect(), execute Connect() again to establish a connection.
