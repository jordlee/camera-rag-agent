.. _command-TrackingOnAndAFOn:

CrCommandId_TrackingOnAndAFOn
=============================================

Control tracking on + AF on button.

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - CrCommandParam_Up
    - | Be sure to specify "Up" after specifying "Down".

  * - CrCommandParam_Down
    - | Specify "Down" when you want to enable both Tracking and AF.
      | Valid when :ref:`CrDeviceProperty_TrackingOnAndAFOnEnableStatus<dp-TrackingOnAndAFOnEnableStatus>` is Enable.
