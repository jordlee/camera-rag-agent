.. _command-TimeCodePresetReset:

CrCommandId_TimeCodePresetReset
=============================================

Execute time code preset reset.

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - CrCommandParam_Up
    - | Be sure to specify "Up" after specifying "Down".

  * - CrCommandParam_Down
    - | Specify "Down" when you want to reset Time Code Preset.
      | Valid when :ref:`CrDeviceProperty_TimeCodePresetResetEnableStatus<dp-TimeCodePresetResetEnableStatus>` is Enable.

| This Control Code can be executed only when the device property "Time Code Preset Reset Enable Status" is Enable (0x01).
