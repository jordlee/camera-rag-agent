.. _command-S1andRelease:

CrCommandId_S1andRelease
=============================================

Execute shutter half-release (S1) and shutter release (S2) buttons.

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - CrCommandParam_Up
    - | Up the shutter button

  * - CrCommandParam_Down
    - | Down the shutter button
      | After executing “Down”, send “Up” to cancel the Down status.
