.. _command-CreativeLookReset:

CrCommandId_CreativeLookReset
=============================================

Execute creative look reset.

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - CrCommandParam_Up
    - | Be sure to specify "Up" after specifying "Down".

  * - CrCommandParam_Down
    - | Specify "Down" when you want to reset Creative Look.
      | Valid when :ref:`CrDeviceProperty_CreativeLookResetEnableStatus<dp-CreativeLookResetEnableStatus>` is Enable.

| Reset settings for the creative look selected in “Creative Look.”
| This Control Code can be executed only when the device property "Reset Creative Look Enable Status" is Enabled (0x01).
