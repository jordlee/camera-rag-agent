.. _command-PixelMapping:

CrCommandId_PixelMapping
=============================================

Execute pixel mapping.

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - CrCommandParam_Down
    - | Specify "Down" when you want to image sensor optimization(Pixel Mapping).
      | Valid when :ref:`CrDeviceProperty_PixelMappingEnableStatus<dp-PixelMappingEnableStatus>` is Enable. When the Pixel Mapping is finished, the camera restart.

| Note:
| "Auto Pixel Mapping" is not performed with this function. It is recommended to perform CrCommandId_PixelMapping periodically.
