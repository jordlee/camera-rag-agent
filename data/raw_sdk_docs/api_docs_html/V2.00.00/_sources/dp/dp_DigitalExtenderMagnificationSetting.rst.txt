CrDeviceProperty_DigitalExtenderMagnificationSetting
====================================================

Get/Set the digital extender magnification setting.

    CrDataType: `CrDataType_UInt32Array`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - Any
    - | 1000 times the real value of Digital Extender Magnification.

| e.g.) 0x000003E8 = 1000 (x1.0)
|       0x000005DC = 1500 (x1.5)
|       0x000007D0 = 2000 (x2.0)
|       0x00000FA0 = 4000 (x4.0)
