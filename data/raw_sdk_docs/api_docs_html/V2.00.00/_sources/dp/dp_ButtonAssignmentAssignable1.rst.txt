.. _dp-ButtonAssignmentAssignable1:

CrDeviceProperty_ButtonAssignmentAssignable1
============================================

Get/Set the Button Assignment Assignable 1

    CrDataType: `CrDataType_UInt8Array`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - Variable(0x00 ~ 0xFF)
    - | Assign a certain function to :ref:`CrDeviceProperty_AssignableButton1<dp-AssignableButton1>` so that the function can be executed by button operation.
      | GetValues() contains a list of function-code that can be assigned to :ref:`CrDeviceProperty_AssignableButton1<dp-AssignableButton1>`.
      | Function-code are 8-bit values, and the number (number of functions) varies depending on the model and setting status. You can use GetDisplayStringList() to get a list of assignable function names.
      | See “:ref:`Get the menu display string<op-get-the-menu-display-string>`”
