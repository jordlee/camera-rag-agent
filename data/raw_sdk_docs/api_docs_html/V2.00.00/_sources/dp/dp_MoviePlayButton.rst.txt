CrDeviceProperty_MoviePlayButton
================================

Get/Set the Movie Play button

    CrDataType: `CrDataType_UInt16Array`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - CrMovieXButton_Up
    - | Be sure to specify "Up" after specifying "Down".

  * - CrMovieXButton_Down
    - | Specify "Down" when you start movie play.
