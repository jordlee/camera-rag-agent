CrDeviceProperty_SelectUserBaseLookToSetInPPLUT
===============================================

Set/Get to Select UserBaseLook to Set in PPLUT

| Can only be set in movie mode.

    CrDataType: `CrDataType_UInt16Array`

  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation
