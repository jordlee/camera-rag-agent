CrDeviceProperty_BaseLookValue
==============================

Get/Set the BaseLook value.

  CrDataType: `CrDataType_UInt16Array`

| It may increase or decrease because it varies depending on the model and setting status.
| Refer to :ref:`GetDisplayStringList<api-get-display-string-list>` for the display character.
| If a BaseLook is extracted to the temporary space, the BaseLook index value including the temporary offset is added to the enumeration list.
| If a BaseLook is removed from the temporary space, it is removed from the enumeration list.
| The offset of Temporary is 0x8000. The next 4 bits are the offset of the Baselook used in Temporary. The lower 16 bits are the index of the Baselook.

| For example, when you recall SceneFile Index1, the BaseLook Index of SceneFile Index1 is set to Temporary.
| The Value is 0x8201 (Temporary offset 0x8000, BaseLook offset 0x0200, Index1 0x01).
| Get the list of value with GetDisplayValueSize and GetDisplayValues.
