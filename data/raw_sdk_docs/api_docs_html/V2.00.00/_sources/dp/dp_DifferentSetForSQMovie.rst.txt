CrDeviceProperty_DifferentSetForSQMovie
=======================================

Get/Set the Different Set for S&Q/Mv

    CrDataType: `CrDataType_UInt8Array`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - CrDifferentSetForSQMovie_On_Different
    - | ON (Different)

  * - CrDifferentSetForSQMovie_Off_Identical
    - | OFF (Identical)

| If this command is Different, Movie/S&Q commands are independent.
| If this command is Identical, Movie command will be common, and S&Q command will be Disable.
| Affected commands:   
|   :ref:`CrDeviceProperty_Movie_Recording_FrameRateSetting<dp-Movie_Recording_FrameRateSetting>`
|   :ref:`CrDeviceProperty_SQRecordingFrameRateSetting<dp-SQRecordingFrameRateSetting>`
|   :ref:`CrDeviceProperty_Movie_Recording_Setting<dp-Movie_Recording_Setting>`
|   :ref:`CrDeviceProperty_SQRecordingSetting<dp-SQRecordingSetting>`
