CrDeviceProperty_Media_FormatProgressRate
=========================================

Get the Media Format Progress Rate

    CrDataType: `CrDataType_UInt32Range`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Value
    - Explanation

  * - 0x00000000
    - Invalid

  * - Other than above values
    - Progress rate

| Calculate the progress rate each time.
| e.g.) 0x003600C8 means 27%. (using the following calculations: (0x36 / 0xC8) * 100)
