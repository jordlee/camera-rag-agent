CrDeviceProperty_VideoStreamMovieRecPermission
==============================================

Get/Set the streaming movie recording permission.

  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - CrVideoStreamMovieRecPermission_NotPermitted
    - | NotPermitted

  * - CrVideoStreamMovieRecPermission_Permitted
