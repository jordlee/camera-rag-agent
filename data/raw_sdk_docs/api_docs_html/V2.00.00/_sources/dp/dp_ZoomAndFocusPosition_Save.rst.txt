CrDeviceProperty_ZoomAndFocusPosition_Save
==========================================

Get/Set the Save Zoom&FocusPosition Preset.

    CrDataType: `CrDataType_UInt8Array`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - 0x00 ～ 0xFF
    - | Save preset number


| The current focus position, Optical Zoom position (Power Zoom lens only), and lens information are stored in the specified preset number.
| With :ref:`CrDeviceProperty_ZoomAndFocusPosition_Load<dp-ZoomAndFocusPosition_Load>`, you can get the saved preset data and restore it to the same state.
| Stored preset data will not be deleted even after initializing the camera. If you specify a preset number that is already in use, that preset number will be overwritten with the new preset data.
| e.g.)
|   {0x00,0x01,0x02} means numbers 0 to 2 can be used
