CrDeviceProperty_Movie_HDMIOutputAudioCH
========================================

Get/Set the Audio Output HDMI Monitor CH 

    CrDataType: `CrDataType_UInt16Array`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - CrHDMIOutputAudioCH_CH1CH2
    - | CH1/CH2

  * - CrHDMIOutputAudioCH_CH3CH4
    - | CH3/CH4

| Audio output channels are assigned in order from the lowest bit. 
| CH1 is the 0th bit, CH2 is the 1st bit, CH3 is the 2nd bit, and CH4 is the 3rd bit.
| Examples:
| When setting CH1/CH2, the 0th and the 1st bit are 1, which is 0x0003.
| When setting CH3/CH4, the 2nd and the 3rd bit are 1, which is 0x000C.
