CrDeviceProperty_MoviePrevButton
================================

Get/Set the Movie Prev button

    CrDataType: `CrDataType_UInt16Array`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - CrMovieXButton_Up
    - | Moves to the top of the previous movie.

  * - CrMovieXButton_Down
    - | The specifications of this device property are the same as :ref:`CrDeviceProperty_MoviePlayButton<dp-MoviePlayButton>`.
