.. _dp-RecordablePowerSources:

CrDeviceProperty_RecordablePowerSources
=======================================

Get all power source kinds on which the recording function is available.

    CrDataType: `CrDataType_UInt8Array`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - CrRecordablePowerSources_DC
    - | DC

  * - CrRecordablePowerSources_Battery
    - | Battery

  * - CrRecordablePowerSources_PoE
    - | PoE

| The supported values of this device property tell on which power source your camera can record internally.
| Refer to the device property ":ref:`CrDeviceProperty_PowerSource<dp-PowerSource>`" to know the current power source.
