CrDeviceProperty_ValidRecordingVideoFormat
==========================================

Get the valid recording video format.

    CrDataType: `CrDataType_UInt32Array`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Name
    - Type
    - Summary
  
  * - Resolution
    - CrValidRecordingVideoFormat_Resolution
    - | Resolution.
      | bit[0-7]

  * - Bitrate
    - CrValidRecordingVideoFormat_Bitrate
    - | Bitrate.
      | bit[8-15]

  * - Scan Line Type
    - CrValidRecordingVideoFormat_ScanLineType
    - | Scan Line Type.
      | bit[16]

  * - Reserve
    - Reserve
    - | Reserve.
      | bit[17-31]

| e.g) 0x00000001(Progressive None 1920×1080)
| 0x00000101(Progressive 35Mbps 1920×1080)
| 0x00000201(Progressive 50Mbps 1920×1080)
| 0x00000002(Progressive None   3840×2160)
| 0x00000003(Progressive None   4096×2160)
| 0x00000004(Progressive None   1280×720)
| 0x00010001(Interlace None 1920×1080 )
| 0x00010101(Interlace 35Mbps 1920×1080)
| 0x00010201(Interlace 50Mbps 1920×1080)
| 0x00010301(Interlace 25Mbps 1920×1080)
