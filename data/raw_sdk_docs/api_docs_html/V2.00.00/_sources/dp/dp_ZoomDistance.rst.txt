.. _dp-ZoomDistance:

CrDeviceProperty_ZoomDistance
=============================

Get/Set the Zoom Distance

    CrDataType: `CrDataType_UInt32Range`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Value
    - Explanation

  * - Variable
    - min

  * - Variable
    - max

  * - Variable
    - step

| Focal length information. Units of 0.001mm.
| min/max/CurrentValue should be set in units of “step”.
| Ex.) min: 18000, max: 55000, step: 1000, value: 20000 (min = 18mm, max = 55mm, value = 20mm)
| The maximum value as a protocol is 4294967 mm.

| Note:
| Indicates the distance when :ref:`CrDeviceProperty_ZoomDistanceUnitSetting<dp-ZoomDistanceUnitSetting>` is CrZoomDistanceUnitSetting_mm. When :ref:`CrDeviceProperty_ZoomDistanceUnitSetting<dp-ZoomDistanceUnitSetting>` is CrZoomDistanceUnitSetting_percent, refer to :ref:`CrDeviceProperty_Zoom_Bar_Information<dp-Zoom_Bar_Information>`.
