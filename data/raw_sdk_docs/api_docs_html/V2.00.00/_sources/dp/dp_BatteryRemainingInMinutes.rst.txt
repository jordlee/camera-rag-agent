CrDeviceProperty_BatteryRemainingInMinutes
==========================================

Get the Battery Remaining in minutes

    CrDataType: `CrDataType_UInt32Range`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Value
    - Explanation

  * - 0x00000000
    - min

  * - 0xFFFFFFFF
    - max

  * - 0x01
    - step

| Unit is minute.
| CrBatteryRemainingInMinutes_Untaken(0xFFFFFFFF) is untaken.
