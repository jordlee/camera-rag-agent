.. _dp-MoviePlayingSpeed:

CrDeviceProperty_MoviePlayingSpeed
==================================

Get Movie Playing Speed

    CrDataType: `CrDataType_UInt64Array`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - CrMoviePlayingSpeed_Nothing
    - | nothing to display.

  * - Other than above values
    - | The real value of Clip Playing Speed (Upper four bytes: numerator, Lower four bytes: denominator)

| The numerator is int32_t type and the denominator is uint32_t type.
