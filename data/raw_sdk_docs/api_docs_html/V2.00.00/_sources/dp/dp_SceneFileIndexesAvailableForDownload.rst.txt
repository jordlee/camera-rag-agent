CrDeviceProperty_SceneFileIndexesAvailableForDownload
=====================================================

Get the SceneFile Indexes Available for Download.

    CrDataType: `CrDataType_UInt32Array`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - 0x00000000
    - | None

  * - Other than above values
