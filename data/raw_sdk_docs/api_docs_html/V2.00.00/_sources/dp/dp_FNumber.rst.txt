CrDeviceProperty_FNumber
========================

Get/Set the Aperture Value (F-Number)

    CrDataType: `CrDataType_UInt16Array`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - Other values
    - | The value is obtained by multiplying a real FNumber value by 100.
      | e.g.) 0x0190 =400 (means F-4)
      |         0x03B6 = 950 (means F-9.5)

  * - CrFnumber_IrisClose
    - | Iris Close

  * - CrFnumber_Unknown
    - | Display "--"

  * - CrFnumber_Nothing
    - | Nothing to display
