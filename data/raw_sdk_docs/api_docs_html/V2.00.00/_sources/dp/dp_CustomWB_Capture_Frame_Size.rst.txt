CrDeviceProperty_CustomWB_Capture_Frame_Size
============================================

Get the Custom WB Capture Frame Size(x,y)

    CrDataType: `CrDataType_UInt32Range`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Value
    - Explanation

  * - 0x00000000
    - min

  * - 0xFFFFFFFF
    - max

  * - 0x00000001
    - step

| The frame width is set in the upper two bytes and the frame height is set in the lower two bytes
| Note :
|   Enabled by executing :ref:`CrDeviceProperty_CustomWB_Capture_Standby<dp-CustomWB_Capture_Standby>` and changing to standby state. When it changes to Standby state, :ref:`CrDeviceProperty_CustomWB_Capture_Operation<dp-CustomWB_Capture_Operation>` changes to Enable, and the current value of :ref:`CrDeviceProperty_CustomWB_Capture_Standby<dp-CustomWB_Capture_Standby>` changes to Disable.
