CrDeviceProperty_BatteryRemain
==============================

Get the Battery Remaining (%)

    CrDataType: `CrDataType_Int16Range`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Value
    - Explanation

  * - 0xFF (untaken)
    - min

  * - 0x64 (100%)
    - max

  * - 0x01
    - step
