.. _dp-ShutterAngle:

CrDeviceProperty_ShutterAngle
=============================

Get/Set the Shutter Angle

    CrDataType: `CrDataType_UInt32Array`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - CrShutterAngle_Disable
    - | -

  * - Other than above values
    - | 1000 times the real value of Shutter Angle
      | e.g.) 0x0002BF20 = 180000 /1000 = 180
      | e.g.) 0x00015F90 = 90000 /1000 = 90
