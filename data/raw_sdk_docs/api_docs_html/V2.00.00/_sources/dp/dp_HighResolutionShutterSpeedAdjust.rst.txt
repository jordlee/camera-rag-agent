.. _dp-HighResolutionShutterSpeedAdjust:

CrDeviceProperty_HighResolutionShutterSpeedAdjust
=================================================

Execute the high resolution SS adjust.

    CrDataType: `CrDataType_Int16Range`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Value
    - Explanation

  * - -32768
    - min

  * - 32767
    - max

  * - 1
    - step


| This Control Code can be executed only when the device property ":ref:`High Resolution Shutter Speed<dp-HighResolutionShutterSpeed>`" is Enable (0x01).
