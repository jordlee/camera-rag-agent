.. _dp-HighResolutionShutterSpeedAdjustInIntegralMultiples:

CrDeviceProperty_HighResolutionShutterSpeedAdjustInIntegralMultiples
====================================================================

Execute the high resolution SS adjust in integral multiples.

    CrDataType: `CrDataType_Int16Range`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Value
    - Explanation

  * - -1
    - min

  * - 1
    - max

  * - 1
    - step


| This Control Code can be executed only when the device property ":ref:`High Resolution Shutter Speed<dp-HighResolutionShutterSpeed>`" is Enable (0x01).
