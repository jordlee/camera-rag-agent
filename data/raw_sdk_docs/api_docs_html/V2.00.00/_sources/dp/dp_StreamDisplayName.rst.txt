.. _dp-StreamDisplayName:

CrDeviceProperty_StreamDisplayName
==================================

Get/Set the Stream Display Name

    CrDataType: `CrDataType_STR`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Value
    - Explanation

  * - String
    - Setting Name (Min Length: 1, Max Length: 255)


| The Display Names that are attached to all Video Streams can be read out at once by using the :ref:`GetDisplayStringList()<api-get-display-string-list>`.
| The Stream Display Name has string of one or more characters. If any other length is set, it will be ignored.
