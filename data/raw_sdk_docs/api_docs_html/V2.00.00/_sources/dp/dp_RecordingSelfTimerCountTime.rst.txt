CrDeviceProperty_RecordingSelfTimerCountTime
============================================

Get/Set the Movie Recording Self timer Count time

    CrDataType: `CrDataType_UInt16Array`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - CrMovieRecordingSelfTimerCountTime_None
    - | Not counting down

  * - Other than above values
    - | Unit is second
      | e.g.) 0x000A = 10 sec
