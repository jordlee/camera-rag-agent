.. _dp-ColorTuningAB:

CrDeviceProperty_ColorTuningAB
==============================

Get/Set the Biaxial Fine Tuning A-B Direction

    CrDataType: `CrDataType_UInt8Range`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Value
    - Explanation

  * - 0x9C (B9_00)
    - min

  * - 0xE4 (A9_00)
    - max

  * - 0x01 (0.25)
    - step


| AB value sent to PC App from camera corresponds to one of the following patterns.
| AB number is BY or AY, where Y is decimal from 0.00 to 9.00 and increments by 0.25.
| Ex.) B9.00(0x9C),  B8.75(0x9D),  ...,
|        A8.75(0xE3), A9.00(0xE4).
| Note:
| There may be parameter scope differences due to model differences.
