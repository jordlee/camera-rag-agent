CrDeviceProperty_PlaybackContentsGammaType

    CrDataType: `CrDataType_UInt16Array`

  :header-rows: 1

    - | Recorded in S-Gamut3.Cine / S-Log3

  * - CrPlaybackContentsGammaType_BT2020_S_Log3

    - | Recorded under other conditions
