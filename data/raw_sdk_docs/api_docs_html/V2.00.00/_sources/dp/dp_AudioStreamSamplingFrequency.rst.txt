CrDeviceProperty_AudioStreamSamplingFrequency
=============================================

Get/Set the streaming sampling frequency.

    CrDataType: `CrDataType_UInt16Array`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - value
    - | sampling frequency

| The sampling frequency can be calculated as follows
| sampling frequency(Hz) = value /10
