CrDeviceProperty_FocalDistanceInMeter
=====================================

Get/Set the Focal Distance in Meter

    CrDataType: `CrDataType_UInt32Range`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Value
    - Explanation

  * - Variable
    - min

  * - Variable
    - max

  * - Variable
    - step

| 1000 times the real value of focal distance in meters.
| If current value is CrFocalDistance_Infinity(0xFFFFFFFF), ∞.
| e.g.) 0x00005014 = 20500 /1000 = 20.5 meter
| e.g.) 0x00030D40 = 200000 /1000 = 200 meter
