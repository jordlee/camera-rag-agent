CrDeviceProperty_FocusOperationWithInt16
========================================

Execute focus operation by specifying direction and speed.

    CrDataType: `CrDataType_Int16Range`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Value
    - Explanation

  * - -32768
    - min

  * - 32767
    - max

  * - 1
    - step


| This Control Code can be executed only when the device property ":ref:`Focus Operation with INT16 Enable Status<dp-FocusOperationWithInt16EnableStatus>`" is Enable (0x01).
| EX) value 0x0001(slow) 0x7FFF(fast) ： Far focus
|      value 0xFFFF(slow) 0x8000(fast) ： Near focus
|      value 0x0000 ： Stop focus
