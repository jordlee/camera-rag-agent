CrDeviceProperty_DateTime_Settings
==================================

Set the Date and Time

    CrDataType: `CrDataType_UInt64`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Value
    - Explanation

  * - "YYYYMMDDThhmmss.s±hhmm"
    - Offset value from UNIX epoch.


| 64bit value. 
| Specify the time in UNIX time (elapsed time from 1970/01/01 00:00:00).
| The time displayed is linked to the time zone setting of the HostPC. 
| The range depends on the model and firmware. 
| Ex.) when 1609582830 is set 
|          = 2021/01/02 10:20:30(UTC) 
|          = 2021/01/02 19:20:30(Tokyo) 
| Some models cannot be set to UTC. 
| Please set your camera to UTC in the menu setting beforehand. 
| Perform an operational check when using the camera. 

| **Related API**
|   :ref:`IDeviceCallback::OnWarning<callback-interface>`
|   See: :ref:`CrDeviceProperty_DateTime_Settings callback<cb-DateTime_Settings>`
