.. _dp-PriorityKeySettings:

CrDeviceProperty_PriorityKeySettings
====================================

Get/Set the Position Key Setting

    CrDataType: `CrDataType_UInt16Array`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - CrPriorityKey_CameraPosition
    - | Camera position priority
      | (Ex. Mode dial, Drive/Focus mode dial)

  * - CrPriorityKey_PCRemote 
    - | PC Remote setting priority

| This value will be set to 0x00 when the session is closed.
