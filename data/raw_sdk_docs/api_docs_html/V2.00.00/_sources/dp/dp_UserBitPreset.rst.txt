CrDeviceProperty_UserBitPreset
==============================

Get/Set the User Bit Preset

    CrDataType: `CrDataType_UInt32Array`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Value
    - Explanation

  * - 0x00000000
    - min

  * - 0xFFFFFFFF
    - max

  * - 0x00000001
    - step

| Saved to file in LSB First order.
