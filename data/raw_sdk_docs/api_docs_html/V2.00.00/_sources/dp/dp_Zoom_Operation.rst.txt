.. _dp-Zoom_Operation:

CrDeviceProperty_Zoom_Operation
===============================

Execute the zoom operation.

| It may not be possible to operate depending on the model and lens type. refs Zoom Operation / Zoom Scale.

| For models that support :ref:`CrDeviceProperty_Zoom_Speed_Range<dp-Zoom_Speed_Range>`, link with Range(min/max/step) of :ref:`CrDeviceProperty_Zoom_Speed_Range<dp-Zoom_Speed_Range>`.

    CrDataType: `CrDataType_Int8Range`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Value
    - Explanation

  * - -1
    - min

  * - 1
    - max

  * - 1
    - step

| This Control Code can be executed only when the device property “Zoom Operation Enable Status” is Enabled (0x01).
| The parameter range is variable. Refer to the device property “Zoom Speed Range” for parameter range.
| Value examples:
| 1: Tele zoom (zoom speed=1)
| -1: Wide zoom (zoom speed=1)
| 0: Stop zoom
