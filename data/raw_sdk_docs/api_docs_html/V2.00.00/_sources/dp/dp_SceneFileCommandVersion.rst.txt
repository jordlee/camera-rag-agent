CrDeviceProperty_SceneFileCommandVersion
========================================

Get the enabled status and dataset version (SceneFile).

    CrDataType: `CrDataType_UInt32Array`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - variable
    - | Dataset Version

| ex) The Dataset Version is 0x00000064 (0d100).
| When SceneFile feature is invalid, IsEnable field is disabled.
