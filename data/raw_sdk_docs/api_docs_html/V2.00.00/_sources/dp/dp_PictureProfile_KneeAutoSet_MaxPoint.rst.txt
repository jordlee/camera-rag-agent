CrDeviceProperty_PictureProfile_KneeAutoSet_MaxPoint
====================================================

Get/Set the Picture Profile Knee AutoSet MaxPoint

    CrDataType: `CrDataType_UInt16Array`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - CrPictureProfileKneeSetPoint_Invalid
    - | Invalid

  * - Other than above values
    - | 100 times the value of MaxPoint(%)
      | ex) 0x2616 = 97.50%

| It may increase or decrease because it varies depending on the model and setting status.
| Set Knee AutoSet MaxPoint for the Picture Profile selected in “Picture Profile.”
