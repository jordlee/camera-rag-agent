CrDeviceProperty_NDFilterOpticalDensityValue
============================================

Get/Set the ND Filter Optical Density Value

    CrDataType: `CrDataType_UInt16Array`

.. list-table::
  :widths: auto
  :header-rows: 1

  * - Parameter Code
    - Explanation

  * - CrNDFilterOpticalDensityValue_Nothing
    - | Nothing to display

  * - Other than above values
    - | 100 times the real value of ND Filter Value
      | e.g.) 0x00D2 = 210/100 = 2.1

| This value expressed in optical density.
